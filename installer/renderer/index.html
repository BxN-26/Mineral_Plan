Bienvenue

Avant de commencer, préparez les informations suivantes.

🌐
Adresse de l'application

Un nom de domaine (planning.monclub.fr) si votre serveur est sur internet, ou l'adresse IP de votre serveur pour un usage en réseau local.

⚠️ Si vous utilisez un nom de domaine, il doit déjà pointer vers ce serveur en DNS avant de continuer (vérifiable chez votre hébergeur).
👤
Compte administrateur

Votre prénom, nom, adresse e-mail et un mot de passe (minimum 8 caractères) pour le premier compte administrateur de l'application.

📧
Email SMTP optionnel

Les identifiants d'une messagerie professionnelle pour permettre la réinitialisation de mot de passe par email. Vous pourrez configurer cela après l'installation.

⏱
Durée estimée : 3 à 10 minutes

L'installation télécharge et compile les dépendances via npm. Une connexion internet est requise.

Type de déploiement

Choisissez comment sera accédée votre application.

🌐
Serveur public (HTTPS)

Votre serveur est accessible depuis internet. Caddy sera configuré pour le HTTPS automatique via Let's Encrypt.

Recommandé
🏢
Réseau local / Intranet

L'application sera accessible uniquement depuis votre réseau interne (LAN). Accès par adresse IP ou nom d'hôte local.

💻
Poste local (test)

Installation sur cette machine uniquement. Accessible via localhost. Idéal pour tester ou développer.

Configuration

Répertoire d'installation et adresse réseau.

L'application sera copiée dans ce répertoire.
Le domaine doit déjà pointer vers ce serveur en DNS.
Port interne de l'API Node.js (défaut : 3000).

Compte administrateur

Ces informations créeront le premier compte de l'application.

Email SMTP optionnel

Permet l'envoi automatique de notifications et de liens de réinitialisation de mot de passe. Vous pouvez ignorer cette étape et configurer cela plus tard.

ℹ️ Cette configuration pourra être modifiée dans Paramètres → Email depuis l'interface de l'application.
